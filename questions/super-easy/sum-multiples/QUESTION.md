# Sum Multiples

**Difficulty:** easy

## Problem

Given a positive integer `n`, return the sum of all numbers from `1` to `n`
that are divisible by `3`, `5`, or `7`.

## Examples

```
Input: n = 7
Output: 21
Explanation: 3 + 5 + 6 + 7 = 21.
```

```
Input: n = 10
Output: 40
```

## Constraints

- 1 <= n <= 1000

## Pattern note

Simple iteration: check each value once and add it if any divisibility test
passes.
