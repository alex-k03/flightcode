# To Lower Case

**Difficulty:** easy

## Problem

Given a string `s`, return the same string with every uppercase English letter
converted to lowercase. Other characters should remain unchanged.

## Examples

```
Input: s = "Hello"
Output: "hello"
```

```
Input: s = "LOVELY"
Output: "lovely"
```

## Constraints

- 1 <= len(s) <= 100
- `s` consists of printable ASCII characters

## Pattern note

String conversion: Python's built-in lowercase conversion handles the required
ASCII letters directly.
